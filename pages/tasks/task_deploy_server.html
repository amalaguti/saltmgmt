What is Salt

Salt is a powerful remote execution and configuration management tool. It allows you to manage and automate the configuration of your servers. The installation guide provides detailed instructions on how to install Salt on various operating systems, including prerequisites, installation steps, and post-installation configuration. Whether you are using Salt for the first time or setting up a new environment, the guide covers everything you need to get started.

How to install Salt

Read more on the Salt Installation Guide

There are three common ways to install Salt on a master server:

  • Install Salt using the Salt multiplatform bootstrap script, simple and provides multiple options, ideal for automated provisioning processes
    • $ curl -o bootstrap-salt.sh -L https://bootstrap.saltproject.io $ chmod +x bootstrap-salt.sh # Install master and minion, version stable 3007.1 $ ./bootstrap-salt.sh -M stable 3007.1
    More info on multiplatform bootstrap-installation
  • Install Salt using the Salt package(s) for your selected platform, simply add the repo and use package manager to install it. Follow the instructions for your target platform
    • # Example for CentOS Stream 9 x86_64 (latest onedir) # Import the SaltStack repository GPG key and add the repository $ sudo rpm --import https://repo.saltproject.io/salt/py3/redhat/9/x86_64/SALT-PROJECT-GPG-PUBKEY-2023.pub $ curl -fsSL https://repo.saltproject.io/salt/py3/redhat/9/x86_64/latest.repo | $ sudo tee /etc/yum.repos.d/salt.repo # Install and then enable/start the Salt master and minion packages $ sudo yum install salt-master $ sudo yum install salt-minion $ sudo systemctl enable salt-master && sudo systemctl start salt-master $ sudo systemctl enable salt-minion && sudo systemctl start salt-minion
    Installation instructions by operating system
  • Install Salt using pip, python platform agnostic installation, commonly used in dev or custom installations to install Salt in Python virtual environments
    • $ pip3 install salt==3007.1
    Platform agnostic installation(pip)
