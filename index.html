What is AmalgaMate?

Salt AmalgaMate
is a full-stack solution that seamlessly integrates various frontend and backend technologies centered around the open Salt project tool.

In case you wonder, the term "AmalgaMate" reflects the amalgamation of these technologies. This tool is designed to be a helpful companion for managing and monitoring Salt projects combined with other technologies.

Purpose

The goal is to provide a simple to use UI on top of Salt,
...yet effective and educational, while demonstrating multiple feasible integrations

This project serves as my experimental platform to explore various technologies and enhance my expertise as a DevOps engineer.
Not a fancy UI designer, just effective enough to get the job done.

The tool is designed to help manage and monitor "Salty" environments, but also as a platform to test other technologies and integrations

In addition it can be used also as a learning or training content portal, which is embedded in the project to help others understand the technologies used in this project and how to execute Salt jobs in a simple manner.

How it works

The tool operates by having the user interface (client) communicate with a middle layer (FastAPI), which exposes various endpoints offering different functionalities. These endpoints primarily interact with the Salt API to execute jobs and return the results to the client. Simple... üòä
